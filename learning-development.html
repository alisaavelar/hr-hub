Learning & Development

Development plans, career pathing, skills gap analysis, and training recommendations — helping every person grow.
1 Skill Cowork Plugin

Why This Matters

Growth is the number one reason people stay — and the number one reason they leave when it's absent. L&D doesn't require a dedicated team or a massive budget. It requires clarity: what does this person want to develop, what does the org need, and how do they intersect? This skill helps build that clarity into actionable plans.

What This Skill Does

📚
Development Plans
Co-create an individual development plan (IDP) with goals, milestones, learning resources, and a timeline — built around the person's role and aspirations.
🗺️
Career Pathing
Map what the next 1–3 career moves look like for a given role — both vertical (promotion) and lateral (cross-functional growth) — with the skills required for each.
🔍
Skills Gap Analysis
Compare where someone is today against where they need to be — for their current role, a promotion, or a new direction — and surface the specific gaps to close.

Individual Development Plan Framework

🎯
Short-term (0–6 months)
Specific skills to develop now. Concrete actions: courses, projects, stretch assignments, or mentoring relationships to pursue in the next two quarters.
🔭
Long-term (6–18 months)
Where this person wants to be and what needs to be true to get there — role changes, leadership responsibilities, or domain expansion.

Trigger Phrases

"Build a development plan for [name]" "What does a career path look like for a [role]?" "Skills gap analysis for [name] who wants to become a [target role]" "What training should we prioritize for the eng team?" "Help [name] write their IDP"

Installation

⚡ Install the HR Hub Plugin

  1. Download hr-hub.plugin
  2. In Cowork, open Settings > Plugins
  3. Click Install from file and select the downloaded .plugin file
  4. All HR skills appear immediately. No restart needed.
