Policies & Processes

Find any policy instantly, document new processes, and keep SOPs current — so nothing lives only in someone's head.
1 Skill Cowork Plugin

Why This Matters

Policies buried in a Google Drive folder nobody looks at might as well not exist. And processes that live in someone's memory disappear when that person moves on. This skill makes policies findable and understandable, and helps document processes so the team can operate consistently — whether it's a new hire doing it for the first time or a veteran doing it their own way.

What This Skill Does

📋
Policy Lookup & Explanation
Instant plain-language answers to any policy question. Covers all HR policies — time off, remote work, expenses, leave, conduct, and more.
📝
Process Documentation
Turn a verbal explanation into a written SOP. Describe a process and get a structured, step-by-step doc ready to share with the team.
🔄
Policy Updates
Redline and update existing policies for clarity, compliance, or org changes — with tracked changes and a plain-language summary of what changed.
🆕
New Policy Drafts
Draft a new policy from scratch — parental leave expansion, remote work guidelines, AI use policy — structured, compliant, and readable.

Installation

⚡ Install the HR Hub Plugin

  1. Download hr-hub.plugin
  2. In Cowork, open Settings > Plugins
  3. Click Install from file and select the downloaded .plugin file
  4. All HR skills appear immediately. No restart needed.
