Recruit & Hire

Interview prep, pipeline tracking, and offer letters — structured, fast, and consistent.
3 Skills Cowork Plugin Stage 01 of 04

Why This Matters

Hiring mistakes are expensive — and most come from inconsistent interviews, informal pipelines tracked in spreadsheets, and offer letters built from a stale Google Doc template. These three skills replace all of that. Every candidate gets the same structured experience. Every offer lands in minutes, not hours.

Three Skills in This Stage

🎤
Interview Prep
Competency-based question banks, structured scorecards, and role-specific interviewer guides — ready in minutes for any position.
🔁
Recruiting Pipeline
Track every active candidate in plain language. Know exactly where each person is, what's next, and what's stalled.
📝
Draft Offer
Generate complete, professional offer letters with total comp breakdown. Base, equity, signing bonus, and start date — all in one pass.

Trigger Phrases

Say any of these in Cowork to activate the skills:

Recruiting Pipeline "What's our pipeline for the PM role?" · "How many candidates are in final rounds?" · "Recruiting update" · "Who's scheduled this week?"

What You Get from Interview Prep

❓
Structured question bank. Behavioral, situational, and role-specific questions organized by competency. Each question includes what a strong answer looks like.
📋
Scorecard with rating rubric. A consistent rubric so every interviewer grades the same competency the same way. Reduces bias and improves calibration.
🧭
Interviewer guide. Who asks what, in which round. A clear interview panel structure so candidates get a consistent, professional experience.

Installation

⚡ Install the HR Hub Plugin

  1. Download hr-hub.plugin
  2. In Cowork, open Settings > Plugins
  3. Click Install from file and select the downloaded .plugin file
  4. All 9 HR skills appear immediately. No restart needed.
